OBJETIVO:
Configurar el proyecto con las CUENTAS y CLAVES del cliente.
Este repositorio NO incluye secretos.

ANTES DE EMPEZAR
[ ] Flutter instalado (flutter --version)
[ ] Node.js instalado (node -v) para Firebase CLI
[ ] Acceso a una cuenta Google (Firebase/Google Cloud)

FIREBASE (OBLIGATORIO)
[ ] He creado un proyecto en Firebase Console
[ ] He activado Firestore Database

FIREBASE CLI (OBLIGATORIO PARA RULES/INDEXES)
[ ] He instalado firebase-tools: npm i -g firebase-tools
[ ] He hecho login: firebase login
[ ] He vinculado el proyecto: firebase use --add
[ ] He desplegado reglas: firebase deploy --only firestore:rules
[ ] He desplegado índices: firebase deploy --only firestore:indexes

ARCHIVOS FIREBASE (NO SE SUBEN A GITHUB)
ANDROID
[ ] google-services.json está en android/app/google-services.json

iOS (si aplica)
[ ] GoogleService-Info.plist está en ios/Runner/GoogleService-Info.plist

macOS (si aplica)
[ ] GoogleService-Info.plist está en macos/Runner/GoogleService-Info.plist

FIREBASE OPTIONS (OBLIGATORIO)
[ ] Copié lib/firebase_options_local.example.dart -> lib/firebase_options_local.dart
[ ] Rellené todos los campos con datos del cliente

GOOGLE MAPS (SI SE USA MAPA)
ANDROID
[ ] Maps SDK for Android activada en Google Cloud
[ ] API Key restringida (Android: package + SHA-1)
[ ] Puse GOOGLE_MAPS_API_KEY en android/local.properties

EJECUCIÓN
[ ] flutter clean
[ ] flutter pub get
[ ] flutter run

SEGURIDAD
[ ] Confirmé que NO subí a GitHub:
    - android/local.properties
    - android/app/google-services.json
    - ios/Runner/GoogleService-Info.plist
    - macos/Runner/GoogleService-Info.plist
    - lib/firebase_options_local.dart
