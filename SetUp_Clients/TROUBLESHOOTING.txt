========================================
TROUBLESHOOTING.txt
========================================

1) ERROR: Missing google-services.json (Android)
- Falta android/app/google-services.json
- Solución:
  Descárgalo desde Firebase (Android app) y colócalo ahí.

2) ERROR: FirebaseOptions / DefaultFirebaseOptions not found
- Falta lib/firebase_options_local.dart o está vacío
- Solución:
  Copiar lib/firebase_options_local.example.dart -> lib/firebase_options_local.dart
  Rellenar con valores desde Firebase Console.

3) ERROR: PERMISSION_DENIED / Security rules block requests
- El cliente NO desplegó reglas o no coinciden con el proyecto.
- Solución:
  npm i -g firebase-tools
  firebase login
  firebase use --add
  firebase deploy --only firestore:rules

4) ERROR: Firestore index required (FAILED_PRECONDITION)
- Falta el índice necesario.
- Solución:
  firebase deploy --only firestore:indexes
  Esperar a que Firestore cree el índice (puede tardar minutos).

5) ERROR: API key not authorized / Maps not loading (Android)
- Restricciones incorrectas en Google Cloud
- Solución:
  Verificar:
  - Package name correcto
  - SHA-1 correcto
  - API habilitada: Maps SDK for Android
  - API restriction aplicada a la key

6) "AdvancedMarkers... Capabilities unavailable without a Map ID"
- No es error: funciones avanzadas requieren Map ID.
- Solución opcional:
  Crear Map ID en Google Cloud (Maps) y usarlo si el proyecto lo soporta.
